Secure Admin Access

Exclusive Access

The system is designed for a single, dedicated administrator. There are no public account creation options or multi-user functionalities, ensuring only authorized personnel can manage hotel operations.

Robust Security

Login is secured with server-side authentication and bcrypt password hashing, protecting your credentials. Rate limiting is implemented to prevent brute-force attacks, and sessions are managed securely with an 8-hour expiration.

Comprehensive Admin Dashboard

Inventory Management

View, add, edit, and delete all hotel items, including tents, utensils, catering equipment, and more. Keep track of quantities and detailed descriptions to maintain accurate stock levels.

Rental Logging

Effortlessly record new rentals with essential details such as item name, quantity hired, renter's name (optional), hire date, expected return date, and price paid. The system automatically adjusts inventory levels.

Performance Reports

Access insightful daily and weekly reports on rental activity and revenue. These reports provide a clear overview of your operations, helping you make informed decisions.

Technical & Security Highlights

Robust Database Integration

Built on a PostgreSQL database, ensuring data integrity and reliability. Rental logging benefits from transaction support, preventing inconsistencies in inventory and rental records.

Vercel Optimized

The system is optimized for deployment on Vercel's free tier, utilizing serverless functions for efficient and cost-effective operation. Static assets are served optimally for fast loading times.
